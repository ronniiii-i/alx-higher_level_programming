Using ``MyList(list)``
-------------------------------

First import ``MyList``

    >>> MyList = __import__('1-my_list').MyList

Now use it:

Basic Usage
===========

Define list and add values:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

Print List:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Print Sorted List:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
